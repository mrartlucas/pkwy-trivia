│ 2. MEDIA LABELING SYSTEM                                                 │
└─────────────────────────────────────────────────────────────────────────┘

PROBLEM: How does system know which graphic/video goes where?
SOLUTION: Tag each file when uploading

MEDIA UPLOAD PROCESS:
  1. Upload file (drag & drop)
  2. Assign Primary Tag:
     • venue_branding → Logo
     • format_intro → Game show intros
     • background → Behind questions
     • sound_effect → Audio feedback
     • winner → Celebration videos
  
  3. Assign to Format (if applicable):
     • Jeopardy
     • Millionaire
     • Family Feud
     • All Formats
  
  4. Set Trigger (for sounds):
     • answer_correct
     • answer_incorrect
     • buzzer_press
     • game_start
     • etc.

EXAMPLE: Upload Jeopardy Background
  ┌────────────────────────────────┐
  │ File: Jeopardy_Background.jpg  │
  │ Tag: background                │
  │ Format: jeopardy               │
  │ → System uses this for         │
  │   Jeopardy questions on TV     │
  └────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ 4. FILE NAMING CONVENTION                                                │
└─────────────────────────────────────────────────────────────────────────┘

FORMAT: {VenueName}_{Type}_{Context}_v{Version}.{ext}

EXAMPLES:
  ✓ PKWY_Logo_Primary_v1.png
  ✓ PKWY_Intro_Welcome_v1.mp4
  ✓ Jeopardy_Background_Blue_v1.jpg
  ✓ Correct_Answer_Chime_v2.mp3

RULES:
  • No spaces (use underscores)
  • Descriptive names
  • Version numbers (v1, v2, v3)
  • Lowercase extensions (.mp4 not .MP4)
